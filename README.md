# Smieciarka App

Aplikacja do oddawania niepotrzebnych przedmiotów w sąsiedztwie.

## Funkcje
- Mapa z przedmiotami do oddania
- Dodawanie własnych przedmiotów
- Czat z właścicielem
- Geolokalizacja

## Tech Stack
- React + Vite
- Supabase (baza danych)
- Leaflet (mapa)

## Uruchomienie
```bash
npm install
npm run dev
```
