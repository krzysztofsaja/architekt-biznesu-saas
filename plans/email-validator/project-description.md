# Opis Projektu: Email Validator & Cleanup Tool

## Problem
Firmy wysyłające newslettery często trafiają na czarne listy spamerów, ponieważ ich bazy zawierają nieistniejące adresy e-mail.

## Rozwiązanie (SaaS)
Automatyczne narzędzie, które sprawdza listę maili i usuwa te błędne, zanim firma wyśle wysyłkę.

## Technologie i Monetizacja
* **Technologie:** Python (backend), React (frontend), baza danych PostgreSQL.
* **Model biznesowy:** Subskrypcja miesięczna lub opłata za liczbę sprawdzonych maili (Pay-as-you-go).
