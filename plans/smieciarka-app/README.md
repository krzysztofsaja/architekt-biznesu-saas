# Projekt SaaS: Śmieciarka App (Zero Waste Community)
**Autor:** Natalia Okołot  
**Kierunek:** Kognitywistyka komunikacji, rok 1 st II  
**Status:** Gotowy do MVP / Do oceny

## 1. Krótki opis projektu
Aplikacja mobilna i przeglądarkowa służąca do oddawania niepotrzebnych rzeczy za darmo. Rozwiązuje problem chaosu na grupach Facebookowych typu "Uwaga, śmieciarka jedzie". Umożliwia szybkie dodawanie zdjęć, geolokalizację przedmiotów oraz bezpośredni kontakt między użytkownikami.
