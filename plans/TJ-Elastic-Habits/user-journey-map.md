# User Journey Map: Adaptive Scaling (Elastic Habits)

**Cel:** Przełamanie paraliżu decyzyjnego poprzez redukcję oporu zadania.

1. **Trigger:** Powiadomienie "Czas na 30 min nauki".
2. **Opór:** Użytkownik czuje przytłoczenie (Executive Dysfunction).
3. **Interwencja:** Aplikacja pyta "Mało energii?" i proponuje wersję Emergency: "Zrób 2 minuty nauki".
4. **Działanie:** Użytkownik akceptuje mniejszą dawkę.
5. **Nagroda:** Zadanie odhaczone, streak utrzymany, brak poczucia winy.

## Wnioski
- Kluczowy moment: Propozycja uproszczenia zadania *zanim* użytkownik je zignoruje.
- Sukces: Utrzymanie nawyku nawet na 1% intensywności.
