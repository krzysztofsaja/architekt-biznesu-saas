# Resource Analysis – NeuroAthlete

## 1. Cel dokumentu

Celem jest określenie zasobów potrzebnych do stworzenia i rozwoju aplikacji NeuroAthlete:

- czasowych  
- ludzkich  
- finansowych  

---

## 2. Założenia

Projekt realizowany jako:

- MVP tworzony przez 1 osobę (student)  
- ograniczony budżet  
- podejście Lean  

---

## 3. Zasoby ludzkie

### MVP (Minimum)

| Rola | Osoba | Zakres |
|------|------|--------|
| Developer | 1 | frontend + logika |
| Product Owner | 1 (ta sama osoba) | decyzje produktowe |
| Tester | użytkownicy | feedback |

👉 Realnie: wszystko robi jedna osoba

---

### Post-MVP

| Rola | Potrzeba |
|------|--------|
| Developer | rozwój funkcji |
| UX Designer | poprawa doświadczenia |
| Marketing | pozyskiwanie użytkowników |

---

## 4. Czas realizacji

### MVP

| Etap | Czas |
|------|------|
| Nauka React Native | 1–2 tygodnie |
| Projekt UI | 3–5 dni |
| Development | 2–4 tygodnie |
| Testy | 1 tydzień |

👉 Łącznie: ~4–8 tygodni

---

### Post-MVP

- rozwój funkcji: ciągły  
- skalowanie: 3–6 miesięcy  

---

## 5. Zasoby technologiczne

### MVP:

- laptop  
- środowisko developerskie  
- emulator telefonu  

---

### Post-MVP:

- Firebase  
- narzędzia analityczne  
- hosting
